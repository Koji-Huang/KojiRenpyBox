# Dammu


很基础的弹幕组件效果, 实现了滚动弹幕效果, 并且保证在弹幕未溢出时不会出现弹幕重叠

<img src="readme.png" height="400" />

---

此组件有两个类

## Danmu

弹幕信息类, 有两个传参

| 参数名 | 数据类型 | 描述 | 默认值 |
| -------------------- | ---- | ---- | --- |
| text | str | 弹幕文本 | "" |
| show_st | None \| float | 弹幕显示时间, 为 None 时会在被添加到弹幕池的时候计时 | None |

## DammuDisplay

弹幕显示类, 参数写死在 dammu_ren.py 里面了, 里面有注释

将此可视化组件添加到屏幕上后, 调用 add_dammu 方法来添加弹幕, 组件会自动显示出此弹幕
